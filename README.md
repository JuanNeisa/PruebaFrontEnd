# PruebaFront

Este proyecto fue construido con [Angular CLI](https://github.com/angular/angular-cli) versión 11.0.6. Es el desarrollo de la prueba de aplicación para Front End.

## Instalacion

Empleando el manejado de paquetes de npm usamos el comando `npm install`.

## Servidor

Dentro del bash ubicado en la carpeta de proyecto usar el comando `ng serve` para desplegar el servidor. Navegue a la dirección `http://localhost:4200/`. La aplicación se podrá actualizar automáticamente a cualquier cambio hecho en los archivos fuerte.
